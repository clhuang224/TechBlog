# Lynn's TechBlog

This repository contains the source code for my personal tech blog, built with [Hexo](https://hexo.io/).

The blog mainly covers notes and write-ups about frontend development, Android, product thinking, UX, workflow habits, and other topics I want to document over time.

Posts are stored in `source/_posts`, and the site is deployed with GitHub Actions.

## Tech Stack

- Hexo
- NexT theme
- pnpm
- GitHub Actions

## Development

Install dependencies:

```bash
pnpm install
```

Start the local server:

```bash
pnpm run server
```

Build the static site:

```bash
pnpm run build
```
